# Appointment System

## Live URL
https://appointment-system-mu.vercel.app

## Description
Ky është një aplikacion për rezervimin e termineve mjekësore online.
Përdoruesit mund të:
- regjistrohen dhe login
- zgjedhin mjekun
- rezervojnë termin
- shohin terminet e tyre

## Technologies
- React
- Supabase (Auth + Database)
- Vercel (Deployment)

## Features
- Authentication (Login/Signup)
- CRUD për termine
- RLS security
- Sugjerim i orëve të lira (AI logic)
